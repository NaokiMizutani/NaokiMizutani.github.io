  • Python3 のインストールに必要なものを先にインストール
  • Python3 のダウンロード
    Python サイト/ダウンロード にて、ダウンロードするpython3 のバージョンを選んで、下記にように /var/tmp などのディレクトリでダウンロード

または

  • ダウンロードした Python3 の解凍と展開、インストール
    （ダウンロードしたファイルが Python-3.5.0.tar.xz の場合）
  • PATH の設定
    /etc/profile または .bashrc に下記の行を追加する。

（.bashrc に追加した場合）追加後、 $ source .bashrc を実行
